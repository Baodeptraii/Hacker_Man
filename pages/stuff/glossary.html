APIs (Application Programming Interfaces)

Là một tập hợp các quy tắc và định dạng cho phép các ứng dụng khác nhau giao tiếp với nhau. Ví dụ: khi bạn sử dụng một ứng dụng trên điện thoại để đăng nhập vào một trang web, ứng dụng đó sẽ gọi đến API của trang web để thực hiện quá trình đăng nhập.

HTTP - HTTPS

HTTP (Hypertext Transfer Protocol): Giao thức truyền tải dữ liệu giữa trình duyệt và máy chủ.

HTTPS (Hypertext Transfer Protocol Secure): Phiên bản bảo mật của HTTP, sử dụng chứng chỉ SSL/TLS để mã hóa dữ liệu truyền tải.

DNS

DNS (Domain Name System) là hệ thống phân giải tên miền cho phép thiết lập tương ứng giữa địa chỉ IP và tên miền trên internet. Khi người dùng nhập địa chỉ trang web trên trình duyệt, DNS sẽ tìm địa chỉ IP của máy chủ chứa trang web và trả về kết quả hiển thị tương ứng của trang web cần tìm.
